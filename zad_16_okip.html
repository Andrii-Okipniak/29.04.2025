Znacznik <div>

Element blokowy, który grupuje inne elementy HTML.

Użycie:

  • Tworzenie sekcji na stronie.
  • Stosowanie stylów CSS.
  • Obsługa JavaScript.
Atrybuty <div>
  • ALIGN – wyrównanie tekstu (przestarzałe)
  • CLASS – przypisanie stylu CSS
  • ID – unikalny identyfikator
  • STYLE – bezpośrednie formatowanie
Znacznik <span>

Element liniowy do wyróżniania fragmentów tekstu.

Użycie:

  • Zmiana wyglądu wybranych słów.
  • Stylizacja tekstu, np. czerwony tekst.
  • Manipulacja tekstem w JavaScript.
Różnice między <div> a <span>
  • <div> – blokowy, organizuje układ strony.
  • <span> – liniowy, służy do stylizacji tekstu.
